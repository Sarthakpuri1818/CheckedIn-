**CheckedIn**

**Overview**
CheckedIn is a modern attendance and workforce management system built to replace outdated manual processes with a smart, automated, and scalable solution.
Designed with simplicity and performance in mind, CheckedIn enables organizations to track attendance, manage approvals, and gain real-time insights  all in one seamless platform

**Features**
Secure Authentication (Login / Logout)
Real-Time Check-In System
Manager Approval & Feedback Workflow
Modern UI with Video Backgrounds
Fast, Responsive, and Scalable Architecture

**TeachStack**
Client(Front-end)
React , NextJs , CSS 

Server(Back-end)
Node.Js , NextJs , REST API's

Database
MongoDB

**Live Demo**

https://checked-in-gamma.vercel.app/



**Workflow**
User logs in (Staff / Manager) -> Staff performs check-in -> Manager reviews check-ins ->
Manager approves/rejects with comments -> Data is updated in real-time
